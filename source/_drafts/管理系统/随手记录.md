- [ ] js的listener，监听keydown延时问题

  - [x] 原因：

    并不是js的问题，而是操作系统的设计就是这样

    就算在txt上长按键盘，也会有停顿。

  - [ ] 解决方法：

    用一个状态变量 + 定时器来实现

    - [ ] 可以实现顺滑调用方法，但是长按时间久了，松开后就会延迟停止

      猜测是每次down都开启一个定时器

      ```js
      let isWriting = false;
      document.addEventListener('keydown', function(e) {
          isWriting = true;
          setTimeout(sayHello);
      })
      document.addEventListener('keyup', function(e) {
          isWriting = false;
      })
      function sayHello() {
          console.log("hello")
          if (isWriting) {
              setTimeout(sayHello);
          }
      }
      ```
